제주도 버스의 효율적인 운행을 위해 퇴근시간 승차인원을 예측하는 모델 개발 프로젝트를 진행했습니다.
데이터를 전처리하고, 파생변수를 생성하며, 랜덤포레스트 모델을 적용하여 예측 모델을 구축하였습니다. 하지만, 예측 모델의 성능이 기대에 못 미치는 결과를 보였습니다.
상관계수 분석 결과 변수들 간에 강한 상관관계가 없어서 변수 선택이 어려웠습니다. 추가적인 데이터 수집과 변수 선택 방법의 개선, 더 정교한 전처리 기술과 모델링 기법의 적용 등을 통해 모델의 성능을 개선해 나가야 겠다고 생각했습니다.
